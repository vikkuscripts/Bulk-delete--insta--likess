## How to Run the Script

Follow the steps below to run the script in your browser using the Instagram desktop website.

### 1. Open Instagram

* Open your browser (Chrome, Edge, or Brave recommended).
* Go to **https://www.instagram.com**.
* Log in to your Instagram account.

### 2. Open Developer Tools

* Press **F12** on your keyboard
  **OR**
* Right-click anywhere on the page and select **Inspect**.

### 3. Open the Console Tab

* In Developer Tools, click the **Console** tab.

### 4. Enable Pasting

Modern browsers block pasting into the console for security reasons.

Type the following command in the console and press **Enter**:

```
allow pasting
```

### 5. Paste the Script

* Copy the JavaScript code from this repository.
* Paste it into the **Console**.
* Press **Enter** to execute the script.

### 6. Follow the Menu

A popup menu will appear where you can select the desired option and run the automation.

---

⚠️ **Important Notes**

* This script works only on the **desktop version of Instagram**.
* Your Instagram interface should be set to **English**.
* Instagram may change its interface at any time, which can break the script.
* Use this script responsibly and at your own risk.
